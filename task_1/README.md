# Лабораторная работа 15 — вариант 3

## Задача
Создать простейший веб‑сервис на Flask с маршрутами:
- `/` — приветствие
- `/hello/<name>` — персональное приветствие
- `/square/<int:number>` — квадрат числа

Индивидуальное задание:
1. `POST /json` — принимает JSON `{name, age}` и возвращает фразу вида `"Привет, {name}! Тебе {age} лет."`
2. `GET /multiply/<a>/<b>` — возвращает произведение двух чисел.
